Included scope

  • List included pages and major sections.
  • List included design deliverables.
  • List included functionality.
  • Define responsive layouts and device coverage.
  • Specify basic SEO, CMS setup, or training if included.

Excluded scope

  • List pages not included.
  • List advanced functionality not included.
  • Clarify copywriting, branding, photography, and legal content responsibilities.
  • Clarify third-party costs and tools.
  • Clarify ongoing maintenance exclusions.

Revision and approval rules

  • Define number of revision rounds.
  • Define what counts as a revision.
  • Define how feedback should be submitted.
  • Set approval deadlines.
  • Explain how late approvals affect timeline.

Change requests and assumptions

  • Define how new features are quoted.
  • Document assumptions about content, access, and availability.
  • Explain what happens if assumptions are incorrect.
  • Confirm client responsibilities.
  • Record sign-off before kickoff.
